# Claude Code 模型配置管理器

> 给完全不懂技术的同事用 😊

---

## Windows 用户

**第一次使用（必做，1 分钟）：**
1. 双击 `启动服务.bat`
2. 浏览器自动打开 → **点击 [新建配置组]**
3. 选「🔥 Groq（完全免费，推荐）」
4. 去 https://console.groq.com 注册，复制 API Key，粘贴进来
5. 点「保存」→ 点「激活」→ 回来按 Enter

**以后每次使用：** 双击 `启动服务.bat` 即可。

---

## Mac / Linux 用户

1. 双击 `start-mac.command`（如果没有反应，在终端运行 `./start-mac.command`）
2. 按提示操作（同上）

---

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 提示"未找到 Node.js" | 下载安装 https://nodejs.org/ |
| 浏览器没自动打开 | 手动访问 http://localhost:3000 |
| Groq 是什么 | 完全免费的模型，不需要充值，注册即用 |
| API Key 在哪找 | 见上方各平台链接 |
| 想换其他模型 | 修改配置 → 点激活，立即生效 |
| 关闭服务 | 关掉黑色终端窗口即可 |
